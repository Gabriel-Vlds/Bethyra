# Bethyra

Sitio web de agencia: maquetación con estética 'Tron Legacy', servicios de desarrollo web y creación de agentes autónomos (Open Claw).

Cómo probar localmente
- Coloca este proyecto dentro de tu servidor Apache (ej. `htdocs/Bethyra`).
- Abre `http://localhost/Bethyra/` en el navegador.

Qué contiene
- `index.php`: HTML principal y secciones (Hero, Servicios, Widgets de muestra).
- `public/assets/css/styles.css`: estilos principales con tema Tron.
- `public/assets/js/app.js`: interacciones de hero y widgets (simulación).

Siguientes pasos sugeridos
- Integrar Open Claw para agentes (backend en `src/api/`).
- Conectar pasarelas reales (Stripe u otro proveedor).
- Mejorar contenidos y añadir formulario de contacto.
